Error API#

Error APIs are used to know the failures caused by any other API used.

Functions#

void alcp_error_str(alc_error_t err, Uint8 *buf, Uint64 size)#

Converts AOCL Crypto errors to human readable form.

This API should be called only if any error occurred while executing other API

Note

API mentions Filename and line number on which error occurred along with error info

Parameters:

  • err – [in] Actual Error

  • buf – [out] Buffer to write the Error message to

  • size – [in] Size of the buffer buf. Size of buf should be at least 1024

bool alcp_is_error(alc_error_t err)#

Returns true if an error has occurred.

This API should be called to check if error has occurred or not

Note

This is the only way to check if an error has occurred in the previous call.

Parameters:

err – [in] Actual Error

void alcp_error_clear(alc_error_t err)#

Clears the error and releases any resources.

This API should be called to Clears the error and releases any resources

Note

At the end of using the error variable, alcp_error_clear() must be called, memory leak would occur otherwise.

Parameters:

err – [in] Actual Error
