RSA API#

RSA algorithm is a public-key cryptosystem. In a public-key cryptosystem, the encryption key is public and decryption key is private. RSA algorithm involves key generation, encryption / decryption and signature.

Data Structures#

pointer to the context of the RSA

alc_error_t alcp_rsa_request(const alc_rsa_key_size keySize, alc_rsa_handle_p pRsaHandle)#

Request a handle for rsa for a configuration as pointed by p_ec_info_p.

Note

Only 1024 and 2048 key size supported

Parameters:

  • keySize – [in] - Supported key size

  • pRsaHandle – [out] - Library populated session handle for future rsa operations.

Error Code for the API called. If alc_error_t is not ALC_ERROR_NONE then alcp_error_str needs to be called to know about error occurred

alc_error_t alcp_rsa_publickey_encrypt(const alc_rsa_handle_p pRsaHandle, alc_rsa_padding pad, const Uint8 *pText, Uint64 textSize, Uint8 *pEncText)#

Function encrypts text using using public key.

This API can be called after alcp_rsa_request

Note

ALCP_RSA_PADDING_NONE is only supported as padding scheme. This has following limitations

  • textSize should equal to the modulus/private_key size

  • pText absolute value should be less than modulus

Parameters:

  • pRsaHandle – [in] - Handler of the Context for the session

  • pad – [in] - padding scheme for rsa encryption

  • pText – [in] - pointer to raw bytes

  • textSize – [in] - size of raw bytes

  • pEncText – [out] - pointer to encrypted bytes bytes

Returns:

Error Code for the API called . if alc_error_t is not zero then alcp_error_str needs to be called to know about error occurred

void alcp_rsa_finish(const alc_rsa_handle_p pRsaHandle)#

Performs any cleanup actions.

This API is called to free resources so should be called to free the session

Note

Must be called to ensure memory allotted (if any) is cleaned.

Parameters:

pRsaHandle – [in] The handle that was returned as part of call together alcp_rsa_request(), once this function is called. The handle will not be valid for future

Returns:

None

alc_error_t alcp_rsa_error(const alc_rsa_handle_p pRsaHandle, Uint8 *pBuff, Uint64 size)#

Get the error string for errors occurring in RSA operations.

This API is called to get the error string. It should be called after alcp_rsa_request and before alcp_rsa_finish

Parameters:

  • pRsaHandle – [in] Session handle for rsa operation

  • pBuff – [out] Destination Buffer to which Error String will be copied

  • size – [in] Length of the Buffer.

Returns:

alc_error_t Error code to validate the Handle
