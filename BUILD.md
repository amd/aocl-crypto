
# AOCL Crypto Library

## Building

$ mkdir build
$ cd build
$ cmake ../


## To enable examples append this
```sh
$ cmake -DALCP_ENABLE_EXAMPLES=1 ../
```


## For Debug build

```sh
$ cmake -DCMAKE_BUILD_TYPE=DEBUG ../
```
Append any other necessary configuration needed for build such as 
`ALCP_ENABLE_EXAMPLES=1` for building examples
